# Hiva Website Redesign - Design Philosophy

## Design Philosophy

### Color Palette
- **Primary**: Deep midnight blue (#0f172a) and soft cream (#f8f9fa)
- **Accent**: Warm coral (#ff6b6b) for energy and motivation
- **Secondary**: Sage green (#6bcf7f) for growth and balance
- **Neutral**: Charcoal (#374151) and light gray (#9ca3af)
- **Gradient**: Subtle aurora gradient from coral to sage for hero backgrounds

### Typography
- **Display Font**: "Tiempos Text" - Bold serif for headlines and hero text
- **Body Font**: "Suisse Int'l" - Clean sans-serif for readability
- **Accent Font**: "JetBrains Mono" - Technical font for code/tech elements
- **Hierarchy**: Large display text (48-72px), medium headings (24-32px), body text (16-18px)

### Visual Language
- **Minimalist Sophistication**: Clean layouts with purposeful white space
- **Organic Shapes**: Subtle curved elements and soft corners
- **Depth & Layering**: Strategic use of shadows and elevation
- **Motion Design**: Smooth, purposeful animations that enhance UX
- **Accessibility First**: High contrast ratios and clear visual hierarchy

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth micro-interactions and element animations
- **Splitting.js**: Advanced text effects and character animations
- **ECharts.js**: Data visualization for user statistics
- **Splide.js**: Smooth carousels for testimonials and features
- **p5.js**: Creative coding for background particle effects
- **Pixi.js**: High-performance visual effects

### Effects & Animations
- **Text Effects**: 
  - Character-by-character reveal animations
  - Gradient text color cycling
  - Split-word stagger effects on scroll
- **Background**: 
  - Subtle particle system with floating geometric shapes
  - Aurora gradient flow animation
  - Depth parallax layers
- **Interactive Elements**:
  - 3D tilt effects on cards
  - Smooth hover transformations
  - Ripple effects on buttons
- **Scroll Animations**:
  - Progressive reveal of content sections
  - Smooth easing curves
  - Staggered element animations

### Styling Approach
- **Mobile-First**: Responsive design starting from 320px
- **Component-Based**: Modular CSS with clear naming conventions
- **Performance Optimized**: Minimal DOM manipulation, efficient animations
- **Dark Mode Ready**: Seamless theme switching with CSS custom properties
- **Touch-Friendly**: Large tap targets, smooth gestures

### Header Effect
- **Floating Navigation**: Semi-transparent backdrop with blur effect
- **Logo Animation**: Subtle breathing effect on hover
- **Menu Transitions**: Smooth slide-in animations for mobile menu
- **Theme Toggle**: Animated sun/moon icon with rotation

### Content Styling
- **Cards**: Soft shadows with hover elevation
- **Buttons**: Gradient backgrounds with smooth transitions
- **Images**: Rounded corners with subtle overlays
- **Forms**: Clean inputs with floating labels
- **Lists**: Custom bullets with accent colors

This design system creates a modern, trustworthy, and engaging experience that positions Hiva as a premium visualization app while maintaining accessibility and performance.
